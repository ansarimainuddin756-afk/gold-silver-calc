# Gold Silver Calculator

Ek simple Progressive Web App (PWA) jo Gold aur Silver ki rates calculate karne mein help karta hai.

## Features
- Gold aur Silver rate calculator
- PWA support (mobile par install kar sakte hain)
- Offline access ke liye service worker

## Files
- `index.html` - Main app page
- `manifest.json` - PWA configuration
- `service-worker.js` - Offline functionality

## Kaise use karein
1. Repository ko clone ya download karein
2. `index.html` ko browser mein open karein
3. Ya GitHub Pages se live version access karein
